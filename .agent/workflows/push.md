## Git Push 到 GitHub

1. 查看目前的變更狀態
// turbo
```bash
git status
```

2. 加入所有變更
```bash
git add .
```

3. 提交變更（請提供 commit message，例如：`git commit -m "feat: 新增商品功能"`）
```bash
git commit -m "<請提供 commit message>"
```

4. 推送到 GitHub
// turbo
```bash
git push origin main
```
